## ğŸ“¦ Installation

### Prerequisites
- .NET 8 SDK
- Node.js (for React dashboard)
- SQL Server (Optional, defaults to SQLite)

### Desktop App
1. Open `TradeMaster.sln` in Visual Studio.
2. Run `Update-Database` in Package Manager Console.
3. Press `F5` to launch.

### Web Dashboard
1. Navigate to `TradeMaster.Web`.
2. Run `npm install`.
3. Run `npm start`.
